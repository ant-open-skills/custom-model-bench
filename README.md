# custom-model-bench

A Claude Code plugin for benchmarking custom Claude agents and prompted workflows against curated datasets with rubric-based judges.

## Install

```
claude plugin marketplace add ant-open-skills/custom-model-bench
claude plugin install custom-model-bench@ant-open-skills
```

Then invoke the skill from any Claude Code session.

## Status

**Phase 0 — scaffolding.** The plugin installs cleanly and exposes the skill, but evaluation logic is not yet implemented. Full workflow (setup / run / judge / report) lands in subsequent phases.
